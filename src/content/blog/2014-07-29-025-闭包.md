```javascript
function f1(){
　　　　var n=999;
　　　　nAdd=function(){n+=1}
　　　　function f2(){
　　　　　　alert(n);
　　　　}
　　　　return f2;
　　}
　　var result=f1();
　　result(); // 999
　　nAdd();
　　result(); // 1000
```
在这个函数中`function f2(){}`，其实是相当于`var f2=function(){}`
在外面我们和可以`nAdd()` 访问`nAdd`这个函数但是我们不能访问f2()这个函数
因为`nAdd=function(){}`   是想一个函数赋给一个全局变量，没有`var`  就是声明的全局变量，闭包没意思，我觉得就是个变量作用域的事
