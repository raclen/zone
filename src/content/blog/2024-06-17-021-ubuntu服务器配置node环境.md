## VPS 服务器部署指南

本指南将带您一步一步地完成在Vultr VPS服务器上部署一个简单Node.js应用的过程。从购买VPS服务器到部署和管理Node.js应用，我们将详细介绍每一个步骤。

### 1. 购买VPS服务器

首先，访问 [Vultr 网站](https://my.vultr.com/) 购买一个VPS服务器。选择合适的配置，根据预算和需求进行选择。例如，每月6美元的套餐适合大部分入门级的应用部署需求。

#### 购买完成后，您将获得以下信息：
- 服务器 IP 地址
- 端口（通常是22，用于SSH连接）
- 登录用户名和密码

### 2. 安装Xshell

Xshell 是一个强大的SSH客户端，适用于Windows系统。下载Xshell并进行安装：

### 3. 使用Xshell连接到VPS

1. 打开Xshell，点击 `文件 > 新建`，然后配置连接信息。
   - **主机地址**: 您的VPS服务器IP地址
   - **端口**: 22（默认SSH端口）
   - **用户名**: 您的VPS服务器用户名
   - **密码**: 您的VPS服务器密码

2. 保存配置后，双击新建的会话连接到服务器。

### 4. 安装FTP工具

在Xshell中安装FileZilla或类似的FTP工具，用于文件传输：

### 5. 更新并安装必要的软件

使用Xshell连接到服务器后，执行以下命令更新软件包并安装必要的软件：

```bash
sudo apt-get update
sudo apt-get install openssl libssl-dev nginx wget git
```

### 6. 安装Nginx并验证安装

Nginx 是一个高性能的HTTP和反向代理服务器，安装完成后，您可以在浏览器中访问您的服务器IP地址（例如：http://149.28.17.xxx/ ），您应该会看到Nginx的欢迎页面。

### 7. 安装Node.js和NVM

1. 使用NVM（Node Version Manager）来安装和管理Node.js版本：

   ```bash
   wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh | bash
   ```

2. 完成安装后，重新打开一个终端窗口或重新连接到服务器，然后安装指定版本的Node.js：

   ```bash
   nvm install 18.12.0
   ```

3. 验证Node.js是否成功安装：

   ```bash
   node -v
   ```

### 8. 部署一个简单的Node.js应用

1. 使用`vi`编辑器新建一个`app.js`文件：

   ```bash
   vi app.js
   ```

2. 按下`i`键进入插入模式，然后输入以下代码：

   ```javascript
   const http = require('http');                                
                                                              
   http.createServer(function(req, res) {                      
       res.writeHead(200, {'Content-Type': 'text/plain'});   
       res.end('八嘎Hentai无路赛');                              
   }).listen(8081);                                            
                                                              
   console.log('server start at 8081');
   ```

3. 按下`ESC`键退出插入模式，输入`:wq!`保存并退出编辑器。

4. 启动Node.js应用：

   ```bash
   node app.js
   ```

### 9. 使用PM2管理Node.js应用

`pm2` 是一个带有负载均衡功能的Node.js进程管理工具，可以保持应用程序在后台运行。

1. 全局安装`pm2`：

   ```bash
   npm install pm2 -g
   ```

2. 使用`pm2`启动应用：

   ```bash
   pm2 start app.js
   ```

3. 停止应用：

   ```bash
   pm2 stop app.js
   ```
