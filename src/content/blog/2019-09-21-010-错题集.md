捕获-目标-冒泡
2. 所有对象都有原型 （错误）
原因：除了基础对象外
3. 对象相等都是false（{name:'aa'}=={name:'aa'}//false）
4. 代码示例
```javascript
function getAge(...args) {
  console.log(args);//[21]
}

getAge(21);
```
5.  下面代码的输出是什么?
```javascript
const a = {};
const b = { key: "b" };
const c = { key: "c" };

a[b] = 123;
a[c] = 456;

console.log(a[b]);//456
//对象键自动转换为字符串[Object object]。
```
6. 一段代码
`new Boolean(false); //true`
7. 下面代码的输出是什么
```javascript
[[0, 1], [2, 3]].reduce(
  (acc, cur) => {
    return acc.concat(cur);
  },
  [1, 2]
);
// [1, 2, 0, 1, 2, 3]
// 这里注意的是第二个参数是默认值，等于函数里面的第一个参数acc
```
8. 下面代码的输出是什么?
```javascript
import myCounter from "./counter";
myCounter++;
console.log("===========",myCounter) //error read-only
```
